# 国際連盟安全環境コンサルティング・ジャパン株式会社 ウェブサイト

## 概要

このプロジェクトは、国際連盟安全環境コンサルティング・ジャパン株式会社の会社案内パンフレットを基に作成された動的なウェブサイトです。

## 特徴

- レスポンシブデザイン（PC・タブレット・スマートフォン対応）
- モダンな UI/UX デザイン
- インタラクティブなタブ切り替え機能
- スムーズなアニメーション効果
- パララックス効果
- ハンバーガーメニュー（モバイル対応）

## 技術仕様

- HTML5
- CSS3（Flexbox、Grid、アニメーション）
- Vanilla JavaScript（ES6+）
- Google Fonts（Noto Sans JP）
- Font Awesome アイコン

## 機能説明

### 1. ヘッダーナビゲーション

- 固定ヘッダーでスクロール時に背景が変化
- スムーススクロール機能
- モバイル対応ハンバーガーメニュー

### 2. ヒーローセクション

- グラデーション背景
- アニメーション付きの 3 つの輪
- パララックス効果

### 3. 会社概要セクション

- カード形式のレイアウト
- ホバーエフェクト付き

### 4. 事業領域セクション

- 事業領域図の表示

### 5. サービスセクション

- タブ切り替え機能
- 6 つのサービス領域（安全、環境、安心、節源、介護・医療、教育）
- フェードインアニメーション

### 6. お問い合わせセクション

- 連絡先情報の表示
- アイコン付きレイアウト

### 7. フッター

- 著作権表示

## 使用方法

### ローカル環境での実行

1. プロジェクトフォルダをダウンロード
2. ターミナルでプロジェクトフォルダに移動
3. 以下のコマンドでローカルサーバーを起動：
   ```bash
   python3 -m http.server 8000
   ```
4. ブラウザで `http://localhost:8000` にアクセス

### ブラウザ対応

- Chrome（推奨）
- Firefox
- Safari
- Edge

## カスタマイズ

### 色の変更

`assets/css/style.css` の以下の部分を編集：

- メインカラー: `#2563eb`
- アクセントカラー: `#10b981`
- 警告カラー: `#fbbf24`

### コンテンツの変更

`index.html` の各セクションを編集してください。

### 画像の変更

`assets/images/` フォルダ内の画像ファイルを置き換えてください。
